# Welcome!

## Core Components

- [Gateway](https://healthchainai.github.io/HealthChain/reference/gateway/gateway/index.md): Connect to multiple healthcare systems and services.
- [FHIR Helpers](https://healthchainai.github.io/HealthChain/reference/utilities/fhir_helpers/index.md): Create, validate, and code FHIR resources with type safety and minimal boilerplate.
- [Agent Tools](https://healthchainai.github.io/HealthChain/reference/utilities/tools/index.md): Typed, validated FHIR tools for agents — served over MCP or LangChain.
- [Sandbox](https://healthchainai.github.io/HealthChain/reference/utilities/sandbox/index.md): A simulated EHR environment — test your service against realistic protocol request flows.
- [Pipeline](https://healthchainai.github.io/HealthChain/reference/pipeline/pipeline/index.md): Compose processing steps around your own models inside protocol flows.
